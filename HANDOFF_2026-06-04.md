# 引き継ぎ書 — 2026-06-04（curio-gen 依頼書「アニメ4種」実装）

curio-gen からの依頼書 [`curio-gen/docs/依頼書_アニメ4種_shorts-script-gen.md`] に沿って、
**アニメーション・プリミティブ4種**を実装した。すべて **preview（TemplateCanvas）と
export（layerComposer / Canvas）で同一レンダラまたは同一数式**で、再生＝書き出しが一致する。

> **curio-gen 担当へ**: 下記「確定 JSON フィールド名・型」に合わせて `[[...]]` ディレクティブを emit してください。
> フィールド名は依頼書の提案から **一部リポ規約に合わせて改名**しています（特に ④）。

---

## 0. 実装ファイル（preview/export ペア）

| 機能 | 型 | preview | export | 共有ロジック |
|---|---|---|---|---|
| ① counter | `Layer.counter` | `renderAnimatedText` | `drawLayerContentInBox`(comment)→`drawText`/`drawAnimatedTextFrame` | `src/lib/counterText.ts` `computeCounterText`/`resolveDynamicText` |
| ② steam | `effect:"steam"` | 書き出しプレビュー Canvas（`renderLayersOnContext`） | `drawLayer` effect 分岐 | `src/lib/effectShape.ts` `drawSteam` |
| ③ flip-swap | `entryAnimation:"flip-swap"` | `computeLayerAnimStyle` + `renderAnimatedText` | `computeCanvasAnim` + テキスト差替 | `resolveDynamicText`（文字差替）、scaleY 数式は両所一致 |
| ④ marker-surge | `shape:"marker-surge"` | `MarkerShapeSvg` | `drawMarkerShape` | `src/lib/markerShape.ts` `computeMarker`（点列＋flash を共有） |

`src/lib/counterText.ts` は **新規ファイル**。①③ の「表示文字列の決定」を 1 か所に集約。

## 1. ① native counter（最優先）— 確定仕様

`comment` レイヤーに `counter` を付けると `layer.text` を無視し、毎フレーム算出した文字列を表示。

```ts
counter?: {
  from: number;
  to: number;
  durationSec: number;          // startSec 起点で from→to にかける秒数（<=0 は無効＝to 静的）
  prefix?: string;              // 数値の前（例 "¥" "$"）
  suffix?: string;              // 数値の後（例 "マルク" "円" "%"）
  separator?: boolean;          // 3桁区切りカンマ（既定 true）
  decimals?: number;            // 小数桁（既定 0）
  ease?: "out" | "linear" | "in" | "inout"; // 既定 "out"
};
```

- 算出: `p = clamp((tSec - startSec)/durationSec, 0, 1)` → ease → `value = from + (to-from)*easedP`。`p>=1` で `to` 固定。
- 整形: `decimals` で四捨五入 → `separator` で3桁区切り（手動・ICU 非依存で決定論）→ `prefix`/`suffix`。
- **停止/編集中・静的 PNG 合成は `to`（最終値）を表示**（draw-on の停止時 p=1 と同じ思想。レイアウト安定）。
- fontSize/fontColor/entry/ambient/textDecoration/charAnimation 等と独立に併用可。
- 受け入れ例: `counter:{from:5000,to:10000,durationSec:3,suffix:"マルク",separator:true}` →
  3秒で「5,000マルク → 10,000マルク」、停止時「10,000マルク」。`decimals:1`+`prefix`、`ease:"linear"` も動作。

> curio の `[[COUNT from=5000 to=10000 | suffix=マルク | dur=3]]` は、**静的レイヤー列 → 単一 counter レイヤー**へ置換するだけ。

## 2. ② steam particles — 確定仕様

`type:"effect"` + `effect:"steam"`（既存 `effect:"particles"/"speedlines"/"spotlight"` と同系統の描画オーバーレイ）。
`effectParams` で制御。決定論（seed=粒子 index・レイヤー生存相対秒）で preview=export 一致。

```ts
effect: "steam";
effectParams?: {
  center?: [number, number]; // 発生位置 %（既定 [50, 82]＝カップ位置あたり）
  count?: number;            // 同時粒子数（既定 24・上限 400）
  color?: string;            // 粒子色（既定 "#FFFFFF"）
  speed?: number;            // 上昇速度倍率（既定 1）
  spread?: number;           // 横揺れ広がり design px（既定 26。上昇に従い広がる）
  rise?: number;             // 上昇量 design px（未指定＝レイヤー高さの約 60%）
  sizeRange?: [number, number]; // サイズ範囲 design px（既定 [10, 26]）
  kind?: "sparkle" | "dust"; // 省略時＝柔らかい湯気（白くゆらぐ上昇）。sparkle=きらめき / dust=細かい塵
};
```

- `[startSec, endSec]` の間、origin から粒子が上へ昇り、広がりながら鐘形 alpha で薄れて消える。
- 受け入れ: カップ位置（例 center [50,60]）に置くと白い湯気がゆらゆら立ち上る。
- **UI なし**（既存の particles/speedlines/spotlight 同様、curio が JSON で emit する前提）。

## 3. ③ flip-to-value（値札フリップ）— 確定仕様

`comment` レイヤーの `entryAnimation:"flip-swap"`。縦に潰れて(scaleY 1→0)、中央で別テキストに差し替え、0→1 で戻す。

```ts
entryAnimation: "flip-swap";
entryDuration?: number; // フリップ全体の秒数（既定 0.3）
flipTo?: string;        // 切替後テキスト
flipAtSec?: number;     // 切替タイミング（startSec 相対秒・既定 entryDuration/2＝scaleY=0 の瞬間）
```

- scaleY = `|p-0.5|*2`（p=0.5＝中央で 0）。transform-origin 中央。preview/export 同一数式。
- 文字差替は `resolveDynamicText`: 再生中は `localTime >= flipAtSec` で `text→flipTo`、停止/静的は `flipTo`。
- 受け入れ: `text:"5,000マルク", flipTo:"10,000マルク", entryAnimation:"flip-swap"` で 5,000 がパタッと潰れて 10,000 に。

## 4. ④ value-surge line（数値サージ）— 確定仕様 ※命名変更あり

依頼書の `shape:"surge-line"` は、既存マーカー注釈基盤（`marker-*` で isMarkerShape 判定・MarkerShapeSvg/drawMarkerShape 共有）に乗せるため
**`shape:"marker-surge"` に改名**しました。`entryAnimation:"draw-on"` で急騰アニメが駆動します。

```ts
shape: "marker-surge";
entryAnimation: "draw-on";     // これで急加速アニメが動く（無いと完成形の静的線）
entryDuration?: number;        // サージ秒数（既定 0.5）
markerFrom?: { x: number; y: number }; // 始点 %（既定 {12, 88}＝左下）
markerTo?:   { x: number; y: number }; // 終点 %（既定 {88, 12}＝右上）
markerHead?: "triangle" | "none";      // 終端ヘッド（既定 "triangle"）
markerWidth?: number;          // 線の太さ design px（既定 6）
markerOvershoot?: number;      // 終端オーバーシュート 線長比 0..0.5（既定 0.1）
fillColor?: string;            // 線色（既定 "#FF3B30"）
```

- 進捗 p（draw-on 線形）を **expo-out** に再マップして一気に描く（イージングは `computeMarker` 内に集約）。
- overshoot ぶん markerTo を突き抜けてから収束。終端に三角ヘッド + **着弾フラッシュ**（描き終わり間際に閃光が咲いて消える）。
- 受け入れ: 画面左下→右上へ線が一気に駆け上がって突き刺さる（curio「パンが3000億倍」の山場用）。

## 5. 型・その他

- `src/types.ts`: `Layer` に `counter?` `flipTo?` `flipAtSec?` `markerOvershoot?` を追加。
  `CounterSpec` 新規。`EntryAnimation` に `"flip-swap"`、`LayerShape` に `"marker-surge"`、
  `DrawnEffectKind` に `"steam"`、`DrawnEffectParams` に `spread?`/`rise?` を追加。
- すべて optional・後方互換維持（指定の無いレイヤーは完全に従来どおり）。
- UI（[LayerPropertyPanel.tsx](src/components/LayerPropertyPanel.tsx)）: ①counter エディタ（テキストセクション）、
  ③flip-swap の切替後/切替秒、④marker-surge の選択肢＋オーバーシュートを追加。②steam は UI なし（JSON emit 前提）。
