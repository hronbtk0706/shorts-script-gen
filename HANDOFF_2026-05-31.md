### A. 単一レンダラ化（CapCut 型）— 最重要の構造変更
プレビュー(DOM/CSS)と書き出し(Canvas)の 2 エンジン体制をやめ、
**「見ているもの＝書き出し（Canvas 単一レンダラ）／操作は裏の DOM」** に統一した。
- [TemplateCanvas.tsx](src/components/TemplateCanvas.tsx): `renderExportFrame`（export 経路
  `renderLayersOnContext` で現在フレームを OffscreenCanvas に描き、可視 canvas に一括 blit）を
  常時前面表示（`const showExport = true`、トグルは撤去済み）。DOM レイヤーは裏で操作専用。
  Moveable ハンドルは `.moveable-control-box { z-index:10000 }`（[App.css](src/App.css)）で前面化。
- 動画/キャラ: プレビューの実 `<video>`/`<canvas>` を `videoFrameSources` に流用（バッファ不要のリアルタイム合成）。
- ドラッグ中: `dragOverrideRef` で掴んだ層の暫定値を Canvas に反映して追従（DOM に切り替えない＝掴む/離すで見た目が変わらない）。
- インライン文字編集中: 編集レイヤーだけ Canvas 描画から除外（`editingLayerIdRef`）し DOM textarea を前面(zIndex 9997)に重ねる。
- 画面エフェクト（shake/zoom/flash/vignette/blur）も renderExportFrame に同コードで適用済み。
- `loadImage` に src キャッシュ追加（毎フレーム再ロード防止）。

### B. preview/export 不一致の一括修正
- C6 text 改行（padding スケール）、motion を export に移植、videoLoop 既定、固定 px の design スケール統一
- text-decoration 複数行（underline-sweep/highlight-bar を行位置基準に）、bubble 枠線/角丸/二重 padding
- 文字縁取り幅: export の `*2` 過剰を除去（preview と一致、細く自然に）
- 和文の縦中央: textBaseline `middle`→`alphabetic` + 字面中心補正 `glyphCenterOffset`（「永」基準）
- 詳細は [MISMATCH_AUDIT_2026-05-28.md](MISMATCH_AUDIT_2026-05-28.md) 末尾。

### D. アニメ仕様 P0（kfs）
curio-gen の仕様書 `curio-gen/docs/shorts-script-gen-animation-spec.md` の P0。
- フィールド名は `kfs`（既存の `keyframes` トラック形式と別系統。衝突回避・後方互換のため）。
- `kfs: AnimKeyframe[]`（startSec 相対秒・per-KF easing・x/y/scale/rotation/opacity）。
- [animKeyframes.ts](src/lib/animKeyframes.ts): easing 8 種 + 補間（指定プロパティのみ/未指定は直前値保持/先頭 ease 無視/最終値保持）。
- kfs 駆動層は entry/exit/motion を抑止（`_kfsDriven`）、ambient は加算で併用可。後方互換（kfs 無しは挙動不変）。
